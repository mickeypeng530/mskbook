# Ankle & Foot Overview

## Imaging Protocol

### Coil & Positioning
- Foot at right angles to lower leg, supine, dedicated surface coil
- Forefoot/toes: prone position (neutral, easier immobilization, decreased magic angle effect)
- Never scan both feet simultaneously (decreased resolution)
- Ankle MRI coverage: extend distally to TMT joints
- Forefoot MRI: extend proximally just past TMT joints

### Plane Terminology
| Plane | Ankle | Foot |
|-------|-------|------|
| **Axial** | Standard | Short-axis (perpendicular to long axis; metatarsals = 5 circles) |
| **Coronal** | Standard | Long-axis (parallel to long axis; like AP radiograph) |
| **Sagittal** | Standard | Standard |

> **⚠️ True sagittal mandatory** for Achilles tendon — oblique cuts create false impression of thickening

### Pulse Sequences
- Routine: T1 + T2 (or STIR/T2*) in all three planes
- Gadolinium: infection, inflammatory arthritis, solid vs cystic mass differentiation

## Best Views Per Structure

| Structure | Best Plane |
|-----------|-----------|
| Achilles tendon | Sagittal |
| Ankle tendons (cross-section) | Axial |
| Tibiofibular ligaments | Axial |
| ATFL / PTFL | Axial |
| Deltoid (deep + superficial) | Coronal |
| Calcaneofibular ligament | Coronal |
| Sinus tarsi | Sagittal + axial + coronal |
| Plantar fascia | Sagittal |

## Systematic Review Checklist
1. **Tendons**: Achilles → medial (Tom Dick And Harry) → lateral (peroneals) → anterior
2. **Ligaments**: Deltoid → lateral (ATFL/CFL/PTFL) → spring ligament → syndesmosis
3. **Plantar fascia**: Thickness (<4 mm normal), signal
4. **Nerves**: Tarsal tunnel contents, intermetatarsal spaces
5. **Bones**: Talar dome (OLT), calcaneus, navicular, sesamoids, coalitions
6. **Soft tissues**: Sinus tarsi fat, retrocalcaneal bursa, masses
